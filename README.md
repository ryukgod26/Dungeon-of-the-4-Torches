# Dungeon of 4 Torches

## Introduction 
This is a Puzzle Game. There are 4 Doors: North Door, West Door, East Door, North Door. The North has a Boss Enemy Behind t and is Locked. You Will need to collect One Key From every other Room to unlock the North Room. Every Door has as Challenge behind it. One Door has a Puzzle where you will have to push a block to a empty space, one room has Slimes and you will have to kill them to get the key and one room you will have to dodge the traps and get the key. This game is in very Early Phase right now because I created this Game in a rush in like 2-3 days. From Now On (3rd feb 2026) i will Start Poliushing this Game.
